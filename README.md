# 🎓 GradeWise – AI-Powered Answer Evaluation System

GradeWise is an AI-powered answer evaluation platform designed to automate the grading of digital and handwritten answer sheets. The system helps educators reduce manual effort, ensure consistent evaluation, and provide constructive feedback to students.

## 🚀 Features

* 📄 Upload answer sheets and submissions
* 🤖 AI-assisted answer evaluation
* 📊 Performance analytics and insights
* 📝 Exam and question management
* 📈 Student score tracking
* 🔐 Secure authentication system
* ☁️ Cloud-based data storage with Supabase

## ⚡ Installation

```bash
git clone <repository-url>
cd gradewise
npm install
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_key
```
